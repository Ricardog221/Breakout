# Breakout
Juego Breakout en p5.js

## Objetivo del Juego
Controla una barra para evitar que la pelota caiga, destruye todos los bloques en pantalla para avanzar de nivel y acumula puntos.

## Controles
- Flecha Izquierda: mover la barra a la izquierda.
- Flecha Derecha: mover la barra a la derecha.
- Barra Espaciadora: para iniciar y pasar de nivel.
